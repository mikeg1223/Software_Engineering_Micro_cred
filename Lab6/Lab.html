What is a Favicon?


When on hears the word favicon it is by no means self explanitory, but in todays digital age it is almost guarenteed that everyone has seen one. Hidden in the top of a browser, it displays in one of several locations. On google chrome you can see it contained in the seperate tab, and while using older browsers like internet explorer, it will be next to the address line itself. So now that we know where it is, you may have already figured it out, huh. It's exactly what you were thinking! It is a small logo-esque image and they are very common. Another spot they show up is next to your browser bookmarks. They're really neat!




How are favicons created?


The act of creating a favicon can be quite a fun one! My suggestion for a website to learn and actively create favicons is: studo.code.org
Throughout this course you will learn how to represent grids of pixels as binary and eventually hexadecimal. For information on binary and hexadecimal I would recommend visiting: This Wiki

How did I create the favicon for this site?

I created this favicon using the website I linked above! The first thing I did was go through the tutorials to gain an understanding of how each character in a bit sequence or hexadecimal sequence affect the color. Next I created the colors I desired inside of my favicon one at a time. Then I chose my favicon to be 20 x 20 pixels, and 24 bit color. Lastly I copy and pasted the hexadecimal values for the colors I was using into the appropriate spots in code which was formatted as a grid for easier reading. And that's it! You can create a favicon just like I did, and have fun with it too!
